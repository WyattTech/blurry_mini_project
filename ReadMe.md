# Mini-Project

## Objective
   * Set an image to load blurry to clear.
   * Set a text to dynamically count down from 0 to 100 as the image comes into focus.
   * Use DOM manipulation to create the blurry effect and create the countdown. 
   * Use filter()
   * Use scale();
